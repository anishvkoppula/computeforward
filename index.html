The honest version

Know what is decided—and what is still being scheduled.

We will not use stale dates, unsupported outcome counters, or invented commitments to push an application.

What is fixed

The program is free, follows one four-level pathway, and uses human admissions review rather than automated decisions.

What comes with an offer

Exact dates, meeting times, time zone, workload, technology requirements, capacity, and named safety contacts.

What “apply” means now

You are joining the next-cohort interest and placement process. No active Summer 2026 cohort is being implied.

One pipeline · four levels

Start where you are. Keep moving forward.

Each level has a distinct purpose. Placement is based on readiness, not age or access to expensive prior instruction.

Level 1 · 8 weeks

Python Foundations

Learn Python and computational thinking by writing, debugging, explaining, and improving real programs.

Best for: absolute beginners. No prior experience is required.

Level 2 · 8 weeks

Applied AI & Machine Learning

Move from knowing Python to using NumPy, pandas, scikit-learn, and introductory neural-network tools in applied mini-projects.

Best for: students with Level 1 skills or baseline Python proficiency.

Level 3 · 10 weeks

Algorithms & Competitive Programming

Practice core data structures, algorithmic patterns, contest strategy, and mock contests aligned with USACO Bronze through Silver readiness.

Best for: intermediate to advanced problem solvers.

Level 4 · ongoing

Passion Project

Develop an independent AI application, startup prototype, developer tool, or other portfolio-grade project with mentor guidance.

Best for: prepared students ready to own a portfolio-grade project.

How it works

Structure, community, progression.

The website is not the classroom. It explains fit, accepts applications securely, and gives families enough information to make a considered decision.

Human placement review

Founders review readiness and follow up. No algorithm makes admissions decisions.

Review the full commitment
